# Biblioteca `fisco-cripto-net.dll`

Biblioteca C# (.NET) para manipulação de certificados digitais e assinatura XML (XAdES/XMLDSig) para integração com sistemas de Documentos Fiscais Eletrônicos (NF-e, NFS-e, CT-e, etc.), em conformidade com o padrão ICP-Brasil.

Biblioteca **`fisco-cripto-net.dll`** em C# (.NET 9.0).
Capaz de carregar certificados digitais e aplicar a assinatura XML Digital (XAdES/XMLDSig).
Um componente essencial para a comunicação com o fisco brasileiro.
